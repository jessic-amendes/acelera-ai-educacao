# Skill de Análise Educacional

Esta pasta contém uma Skill reutilizável para análise de indicadores 
educacionais do World Bank (dataset EdStats).

Veja o arquivo `SKILL.md` para instruções completas de uso.

## Uso rápido

Peça a um agente de IA (Codex, Claude Code) algo como:

> "Usando a skill em skill/SKILL.md, gere um ranking dos 10 países que mais 
> evoluíram no indicador de matrícula secundária"

O agente vai seguir as instruções, boas práticas e limitações documentadas 
na skill, reaproveitando as funções já testadas em `scripts/analises.py`.
